# Homebrew Tap for hwangdev97 Tools

This is a Homebrew tap for tools created by hwangdev97.

## Installation

```bash
brew tap hwangdev97/tools
brew install rosetta
```

## Available Formulae

### Rosetta

A modern iOS localization tool with beautiful CLI.

```bash
brew install rosetta
```

For more information, visit the [Rosetta GitHub repository](https://github.com/hwangdev97/Rosetta).

## Manual Installation

If you prefer not to use Homebrew, you can download the binaries directly from the [releases page](https://github.com/hwangdev97/Rosetta/releases).

## Development

This tap is automatically updated when new releases are published to the main repository.
